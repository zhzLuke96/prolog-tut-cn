# Higher-order Predicates

在Prolog中，高阶谓词（Higher-order predicates）允许我们将谓词作为参数传递给其他谓词，这为编写灵活且强大的程序提供了极大的便利。高阶谓词是Prolog中非常有用的功能，可以用来处理复杂的数据操作和逻辑推理。本章将介绍几个常用的高阶谓词及其应用。

## 何为高阶谓词？

高阶谓词是指接受另一个谓词作为参数的谓词。换句话说，高阶谓词可以接收谓词作为输入，并在其内部调用这些谓词。这使得我们可以将操作抽象化，编写更加通用和复用的代码。

### 常用高阶谓词

#### `maplist/2` 和 `maplist/3`

`maplist/2`和`maplist/3`是高阶谓词中最常用的两个，它们用于将一个谓词应用到列表的每一个元素上。

- `maplist/2`：将一个一元谓词应用到列表的每个元素上。
- `maplist/3`：将一个二元谓词应用到两个列表的每一对元素上。

##### `maplist/3` 示例

假设我们有一个将整数平方的谓词：

```prolog
square(X, Y) :- Y is X * X.
```

我们可以使用`maplist/2`来将`square/2`应用到一个列表的每个元素上：

```prolog
square_list(List, SquaredList) :-
    maplist(square, List, SquaredList).
```

运行示例：

```prolog
?- square_list([1, 2, 3, 4], Result).
Result = [1, 4, 9, 16].
```

##### `maplist/3` 示例

假设我们有两个列表，我们想将它们的对应元素相加。我们可以定义一个加法谓词：

```prolog
add(X, Y, Z) :- Z is X + Y.
```

然后使用`maplist/3`将它应用到两个列表上：

```prolog
add_lists(List1, List2, SumList) :-
    maplist(add, List1, List2, SumList).
```

运行示例：

```prolog
?- add_lists([1, 2, 3], [4, 5, 6], Result).
Result = [5, 7, 9].
```

#### `findall/3` 和 `bagof/3`

`findall/3`和`bagof/3`用于收集满足某些条件的所有解，并将它们放入列表中。

##### `findall/3` 示例

假设我们有一个谓词`parent/2`表示父母和孩子的关系：

```prolog
parent(alice, bob).
parent(alice, carol).
parent(bob, dave).
```

我们可以使用`findall/3`来找到所有的孩子：

```prolog
find_all_children(Parent, Children) :-
    findall(Child, parent(Parent, Child), Children).
```

运行示例：

```prolog
?- find_all_children(alice, Children).
Children = [bob, carol].
```

##### `bagof/3` 示例

`bagof/3`与`findall/3`类似，但它还允许我们在收集结果时进行分组：

```prolog
bag_of_children(Parent, Children) :-
    bagof(Child, parent(Parent, Child), Children).
```

运行示例：

```prolog
?- bag_of_children(alice, Children).
Children = [bob, carol].
```

#### `call/1` 和 `call/2`

`call/1`和`call/2`用于动态地调用谓词。`call/1`接受一个一元谓词，而`call/2`接受一个二元谓词。

##### `call/1` 示例

假设我们有一个谓词`greet/1`：

```prolog
greet(X) :- write('Hello, '), write(X), nl.
```

我们可以使用`call/1`来动态地调用`greet/1`：

```prolog
dynamic_greet(Predicate) :-
    call(Predicate, 'Alice').
```

运行示例：

```prolog
?- dynamic_greet(greet).
Hello, Alice.
```

##### `call/2` 示例

对于二元谓词，例如：

```prolog
add(X, Y, Z) :- Z is X + Y.
```

我们可以使用`call/2`来动态调用它：

```prolog
dynamic_add(Predicate, X, Y) :-
    call(Predicate, X, Y, Result),
    write(Result).
```

运行示例：

```prolog
?- dynamic_add(add, 5, 7).
12
```

## 总结

在这一章中，我们介绍了Prolog中的高阶谓词，包括`maplist/2`、`maplist/3`、`findall/3`、`bagof/3`、`call/1`和`call/2`。高阶谓词为Prolog编程提供了强大的功能，使得我们能够编写更加灵活和复用的代码。掌握这些工具将使你能够处理更加复杂的数据结构和逻辑问题。
