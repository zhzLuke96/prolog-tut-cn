# GUI

> 注意事项：XPCE 从 SWI-Prolog 9.x 起不再捆绑安装。生产项目优先考虑 Web UI 方案（见 c8）。本节的 XPCE 示例仅作概念演示，实际开发建议使用 HTTP + HTML/JS 前端。

> 一句话概括：XPCE 是 SWI-Prolog 自带的 GUI 工具箱——用 `new/2` 创建控件，用 `send/2` 操作，用 `send(@display, ...)` 管理窗口。

> 前置：[Hello world](c1.md) | 难度：★ | 后续：—

## XPCE 概念

XPCE 是 SWI-Prolog 的跨平台 GUI 系统。不是最潮的 UI 框架，但胜在 SWI-Prolog 直接捆绑、无需额外安装。

核心思想：

- 所有 UI 元素是对象（PCE 类）
- `new(+Class, +InitArg)` — 创建对象
- `send(+Object, +Message)` — 发送消息（调用方法）
- `get(+Object, +Attribute, -Value)` — 获取属性

## 基本示例

最简窗口：

```prolog
:- use_module(library(pce)).

simple_window :-
    new(D, dialog("Hello")),
    send(D, append, new(T, text("Hello, XPCE!"))),
    send(D, append, new(B, button("OK", message(@prolog, halt)))),
    send(D, open).
```

运行：

```prolog
?- simple_window.
```

弹出一个窗口，包含文本和按钮，点击 OK 退出。

## new/2, send/2 详解

### new/2

```prolog
new(Dialog, dialog("Title")).       % 对话框
new(Btn,    button("Click")).        % 按钮
new(Txt,    text_item(name)).        % 文本框
new(Lbl,    label("Hello")).         % 标签
new(Img,    image('icon.png')).      % 图像
```

### send/2 和 send/3

```prolog
send(Obj, open).                     % 打开
send(Obj, destroy).                  % 销毁
send(Obj, append, Child).            % 添加子元素
send(Obj, set, X, Y, W, H).         % 设置位置大小
send(Obj, font, font(screen, roman, 14, normal)). % 字体
```

### get/3

```prolog
get(Txt, selection, Value).          % 获取文本框内容
get(Btn, displayed, Value).          % 获取显示状态
get(Dlg, width, W).                  % 获取宽度
```

## 完整示例：计算器

```prolog
:- use_module(library(pce)).

calculator :-
    new(D, dialog("Calculator")),
    new(Input, text_item(number, "")),
    new(Result, label("")),
    new(BtnAdd, button("+", message(@prolog, calc, Input, add, Result))),
    new(BtnSub, button("-", message(@prolog, calc, Input, sub, Result))),
    new(BtnMul, button("*", message(@prolog, calc, Input, mul, Result))),
    new(BtnDiv, button("/", message(@prolog, calc, Input, div, Result))),
    send(D, append, Input),
    send(D, append, Result),
    send(D, append, BtnAdd),
    send(D, append, BtnSub),
    send(D, append, BtnMul),
    send(D, append, BtnDiv),
    send(D, open).

calc(Input, Op, Result) :-
    get(Input, selection, Val),
    atom_number(Val, Num),
    (   Op == add -> Res is Num + Num
    ;   Op == sub -> Res is Num - Num
    ;   Op == mul -> Res is Num * Num
    ;   Op == div -> Res is Num / Num
    ),
    send(Result, selection, Res).
```

启动 `calculator.` 即可使用。（注意：这是演示，不是完整计算器——运算符对同一个数操作。你可以扩展为带寄存器的版本。）

## 事件处理

XPCE 用 `message` 对象绑定事件：

```prolog
%% 点击按钮
send(Btn, message, message(@prolog, my_handler, Arg)).

%% 文本变更时触发
send(Input, message, message(@prolog, on_change, Input)).
```

`@prolog` 引用 Prolog 引擎——XPCE 通过它回调 Prolog 谓词。

```prolog
:- dynamic counter/1.
counter(0).

counter_window :-
    new(D, dialog("Counter")),
    new(Lbl, label("Count: 0")),
    new(Btn, button("+1", message(@prolog, inc, Lbl))),
    send(D, append, Lbl),
    send(D, append, Btn),
    send(D, open).

inc(Lbl) :-
    retract(counter(C)),
    C1 is C + 1,
    assert(counter(C1)),
    send(Lbl, selection, C1).
```

每次点击按钮，计数 +1，标签实时更新。

## 常用控件

| 类            | 用途         | 创建示例                                            |
|---------------|-------------|----------------------------------------------------|
| `dialog`      | 窗口容器     | `new(D, dialog("Title"))`                          |
| `text_item`   | 输入框       | `new(T, text_item(name))`                          |
| `label`       | 标签         | `new(L, label("Text"))`                            |
| `button`      | 按钮         | `new(B, button("OK", message(...)))`               |
| `list_browser`| 列表         | `new(L, list_browser), send(L, append, item)`      |
| `menu`        | 下拉菜单     | `new(M, menu(options)), send(M, append, choice)`   |
| `slider`      | 滑块         | `new(S, slider(min, max, value))`                  |
| `image`       | 图片         | `new(I, image('file.png'))`                        |
| `graphical`   | 画布         | `new(G, picture), send(G, display, Box)`           |

## 画布绘图

```prolog
draw_canvas :-
    new(P, picture("Canvas")),
    new(Box, box(100, 50)),
    send(Box, fill_pattern, colour(red)),
    send(P, display, Box, point(50, 50)),
    new(Circle, circle(40)),
    send(Circle, fill_pattern, colour(blue)),
    send(P, display, Circle, point(200, 50)),
    send(P, open).
```

## XPCE 限制

- 只在 SWI-Prolog 桌面版可用（Windows/Linux/macOS）
- 不支持 Web 或移动端
- 开发体验较老——没有可视化布局工具
- 中文显示需要设置字体

```prolog
:- use_module(library(pce)).
:- initialization
    new(D, dialog("测试")),
    send(D, append, new(T, text("你好，世界！"))),
    send(T, font, font(screen, song, 14, normal)),
    send(D, open).
```

如果中文显示乱码，设置合适的系统字体（`song`, `simsun` 或系统 Unicode 字体）。

## 总结

XPCE 是 SWI-Prolog 自带的 GUI 方案，适合快速原型和内部工具：

- `new/2` 创建控件
- `send/2` 发送操作消息
- `message` 对象绑定事件回调
- 支持 dialog、输入框、按钮、列表、画布等常用组件
- 简单场景比学一套 Web 前端省事——纯 Prolog 搞定前后端
